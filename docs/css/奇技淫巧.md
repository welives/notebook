## 空白折叠

所谓「**空白折叠**」其实是 HTML 代码在编辑器中回车换行后产生的，因为在浏览器渲染时，多个空白符会被合并成一个空格字符进行渲染。空白折叠只会出现在行盒与行块盒排列中
