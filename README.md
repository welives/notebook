# NoteBook

## 说明

`@welives`的笔记文档，依赖于[docsify](https://docsifyjs.netlify.app/#/zh-cn/)构建

## 安装
```shell
npm i -g docsify
```

## 使用
```shell
# 克隆仓库
git clone https://github.com/welives/notebook.git && cd notebook

# 安装依赖
npm i

# 启动服务
npm run docs
```

> 文档`docs`目录的`.nojekyll`文件，是为了防止`github`忽略以下划线开头的文件
